#frontend task student profile Dashboard
A simple and beginner-friendly web project that displays a Student Profile Dashboard using HTML, CSS, and JavaScript.
The project allows a user to enter:
Student Name
Course Name
5 Skills
When the user clicks "Show Profile", a popup box appears displaying all the entered details. The popup can be closed using a Close button.

/project-folder
│── index.html
│── style.css
└── script.js

Features
Front-End (HTML/CSS)

#Clean layout built using Flexbox/Grid
Proper semantic tags: header, section, ul, button
User inputs for:
Student Name
Course Name
5 Skills
A styled popup to display profile details

# JS

Fetches input values using easy-to-understand JS
Shows a popup (NOT an alert box)
Dynamically inserts name, course, and skills into popup
Closes popup when user clicks "Close"


#How It Works
User enters their details in the input fields.
User clicks "Show Profile".
JavaScript collects all the input values.
A popup appears containing the student’s full profile.

#Technologies Used
HTML – For page structure
CSS – For layout, styling, and popup design
JavaScript – For dynamic functionality and popup behavior
Clicking "Close" hides the popup again.


#How to Run the Project
Download all files (index.html, style.css, script.js).
Place them in the same folder.
Open index.html in any web browser (Chrome, Firefox, Edge, etc.).
Fill the form and click Show Profile.
